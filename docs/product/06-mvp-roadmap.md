# 06. MVP Roadmap

## 1. MVP 원칙

MVP에서 너무 많은 기능을 넣으면 앱이 터진다.

특히 목표관리, 루틴관리, 공부관리, 수면관리, 돈관리, AI상담을 모두 MVP에 넣으면 올인원 앱이 아니라 미완성 기능 묶음이 될 가능성이 높다.

따라서 MVP는 다음 흐름에 집중한다.

```text
기록
→ 문제 정리
→ Todo 생성
→ 하루 계획 배치
→ 실행
```

## 2. MVP 필수

```text
회원가입 / 로그인
일기 작성
일기 목록 / 상세 / 수정 / 삭제
회고 작성
정리함 / 문제함 직접 등록
정리함 목록 / 상세 / 수정 / 삭제
Todo 등록
Todo 완료 체크
오늘 해볼 것 보기
하루 계획 직접 생성
시간 블록 직접 생성
Todo를 시간 블록에 연결
```

## 3. MVP 선택

```text
AI 일기 요약
AI 문제 후보 추출
AI 행동 후보 추천
AI 하루 계획 후보 생성
AI 시간표 후보 수정 후 적용
간단한 지출 직접 입력
일기에서 지출 후보 추출
간단한 소비 회고
감정 태그 기반 지출 요약
```

```text
고급 루틴관리
수면관리 고도화
공부관리 고도화
고급 통계
알림

```

### 1단계: 기록과 정리

```text
일기
회고
정리함 / 문제함
Todo
오늘 해볼 것
하루 계획
시간 블록
```

### 2단계: AI 보조

```text

AI 하루 계획 후보 생성
```

### 3단계: 계획 확장

```text
주간 계획
월간 계획

```

### 4단계: 생활 데이터 연결

```text

공부 기록
```
